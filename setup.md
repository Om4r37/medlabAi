# Pre-requisites
- Python 3.10
- Git
# Setup
## Unix
```
sh <(curl -s "https://raw.githubusercontent.com/Om4r37/medlabAi/main/setup.sh")
```
## Windows
```
curl -o setup.ps1 https://raw.githubusercontent.com/Om4r37/medlabAi/main/setup.ps1 && powershell -ExecutionPolicy Bypass -File setup.ps1
```
# Run
```
python run.py
```
